Step by Step Roadmap

上から順に進めれば、そのまま学習計画になる構成です

初学者が途中で迷いやすいポイントを先回りしてつぶすために、「概念」と「操作」を分離しつつ、最後に演習でつなげる形にしています。

1. 概念を先に掴む

同一ネットワーク通信、異なるネットワーク通信、デフォルトゲートウェイ、VLAN、ルーティングの役割を図解で整理します。

2. Packet Tracerに慣れる

機器の置き方、接続、RealtimeとSimulation、PCのIP設定、CLIの開き方を丁寧に確認します。

3. 基礎ラボで成功体験を作る

RouterとSwitchの基本コマンドを学んだあと、PC-SW-Routerの最小構成を作って疎通確認します。

4. 中級ラボとL3 Switchを学ぶ

VLAN間通信、静的ルーティング、SVI による VLAN 間ルーティングを順番に学び、構成の選び方を広げます。

5. 単元末尾で理解度確認

各ページの最後にその単元専用の理解度チェックを置いているので、読んだ内容をその場で確認しながら進められます。

Learning Topics

トピックごとに分けた学習ページ

サイドバーからいつでもジャンプできます。必要なときに概念ページへ戻れるように分割しています。

02. 概念理解 ネットワーク通信はどう成立するのか

PCから送ったデータがSwitchやRouterをどう通るのかを、初学者向けの言葉で説明します。

 03. Packet Tracer 画面の見方と最初の操作

Packet Tracer上で何をどこから触ればいいか、最初に覚えるべき操作だけに絞って案内します。

 04. Cisco CLI show系と設定系の基本コマンド

現場で最初に覚えるCLIを、モード遷移から保存まで順番に学べます。

 05. 基礎ラボ 小さなラボを組んで疎通まで確認

PC、Switch、Routerの最小構成をPacket Tracerで作り、Ping成功まで導きます。

 06. 中級ラボ VLAN間通信と静的ルーティング

Router-on-a-Stick と静的ルートを使って、基礎の次の一歩を練習します。

 07. L3 Switch / SVI SVIでVLAN間通信を学ぶ

Multilayer Switch だけで VLAN 間ルーティングを行い、Router-on-a-Stick との違いも整理します。

 08. トラブルシュート 通信できない時の確認順を固める

Packet Tracerで起きやすい失敗例をもとに、切り分けの順番を整理します。
Goal

この教材で目指す状態

できるようになること

  • 同一セグメント通信と異なるセグメント通信の違いを説明できる
  • Packet Tracerで基本トポロジを自分で組める
  • Router# show ip interface brief や Router# show ip route の役割がわかる
  • RouterとSwitchの最低限の設定を順番通りに入れられる
  • VLAN間通信と静的ルーティングの入口を自力で再現できる
  • L3 Switch の SVI で VLAN 間ルーティングの最小構成を作れる

  • 基礎ラボで手順を固める
  • 中級ラボと L3 Switch / SVI で構成パターンを広げる
  • トラブルシュートで確認順を固める
  • 各単元の最後で理解度チェックを行う
